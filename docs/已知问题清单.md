## 3. 关闭标准

- P2 问题需有执行记录、截图或命令日志。
- 涉及安全、库存、金额和工资的问题必须通过回归测试。
- 涉及正式上线准入的问题必须在上线前关闭；如需降级，必须由业务负责人、技术负责人和运维负责人共同确认。
- 关闭后更新本文件、[验收测试报告](验收测试报告.md) 和 [项目汇总](项目汇总.md)。
