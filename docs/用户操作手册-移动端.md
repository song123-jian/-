# 用户操作手册：移动端

使用 Skill：requirements-doc

## 1. 登录与首页

1. 打开移动端地址。
2. 输入账号和密码登录。
3. 首页展示常用入口和待处理消息。
4. 未登录访问业务页面会跳转到登录页。

## 2. 扫码报工

1. 进入扫码报工。
2. 选择或扫码定位工单。
3. 填写人员、机台、模具、班次、良品数量、不良数量、开始结束时间。
4. 提交后，生产订单完成数和工资日汇总应同步更新。

## 3. 我的产量

查看本人按日期、班次或工单统计的产量记录，用于核对日工资。

## 4. 我的工资

查看本人的日工资、月工资、补贴、扣款和结算状态。发现异常时联系生产主管或财务复核。

## 5. 质检录入

1. 选择工单、产品和检验类型。
2. 填写检验结果、不良类型、不良数量、图片和备注。
3. 不良数量不得超过抽检数量或业务允许范围。

## 6. 库存查询

按产品、仓库、库位、批次查询库存。库存异常时以库存台账和单据为准追溯。

## 7. 盘点录入

1. 选择盘点单。
2. 录入实盘数量。
3. 提交后等待审批。
4. 审批通过后形成差异处理。

## 8. 调拨扫码

1. 扫码或选择待接收调拨单。
2. 核对调出仓、调入仓、产品、批次和数量。
3. 确认接收后，系统写入调拨入库台账。

## 9. 消息通知

查看未读消息，处理后标记已读。重要业务消息需回到对应业务模块复核。

## 10. 待办中心

首页待办中心聚合待报工、待质检、待盘点、待接收调拨、离线重试和注塑专业待办。进入“待办中心”后，可按类型查看任务并跳转到对应页面处理。

## 11. 注塑专业

入口：首页“注塑专业”或 `/m/injection`。

移动端注塑专业覆盖 13 个现场入口：

| 序号 | 入口 | 现场用途 |
| ---: | --- | --- |
| 1 | 工艺卡 | 查看或登记工艺卡基础参数 |
| 2 | 首件确认 | 记录试模和首件结果 |
| 3 | 配料烘料 | 登记原料批次、用量、回料比例、烘料参数 |
| 4 | 批次追溯 | 维护批次上下游追溯节点 |
| 5 | 齐套检查 | 开工前确认原料、模具、机台、首件和人员 |
| 6 | 设备维修 | 上报故障、查看分派、处理和关闭状态 |
| 7 | 模具保养 | 查看待保养、逾期和正常模具计划 |
| 8 | 安灯异常 | 上报、分派、处理、关闭现场异常 |
| 9 | 包装标签 | 维护或查看标签模板和打印数据 |
| 10 | 客诉8D | 上传客诉证据并维护改善信息 |
| 11 | OEE分析 | 录入班次运行数据并计算 OEE |
| 12 | 工艺变更 | 提交工艺、BOM、原料或模具变更申请 |
| 13 | 请购单 | 确认缺料请购建议和请购量 |

操作规则：

1. 选择入口后，页面展示当前模块待处理记录。
2. 点击“新增记录”填写表单；必填项缺失时页面会提示。
3. 记录保存后可在列表中执行“提交、审核、驳回、分派、开始、完成、关闭”等可用动作。
4. 齐套检查失败、安灯异常、维修、首件确认等现场事项会进入待办中心。
5. 网络异常导致提交失败时，优先进入离线任务中心检查可重试任务。
