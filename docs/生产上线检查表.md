# 生产上线检查表

## 1. 基础检查

- [ ] Supabase 项目创建完成。
- [ ] `database/init.sql` 已执行成功。

- [ ] 管理端静态产物发布成功。

- [ ] 回款、费用、对账、工资汇总通过。
- [ ] 库存台账可追溯。

- [ ] P0/P1 缺陷为 0。
